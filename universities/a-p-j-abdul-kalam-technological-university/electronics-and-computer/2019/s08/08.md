# OEERT834: CLOUD COMPUTING AND APPLICATIONS

## course outcomes

- **CO1**: Describe the principles of cloud computing. *(K2)*
- **CO2**: Explain the technologies used in cloud computing and virtualization. *(K2)*
- **CO3**: Describe cloud computing and microservices. *(K2)*
- **CO4**: Describe cloud management and cloud security features. *(K2)*

## syllabus

### Module 1: Cloud Computing Fundamentals
- What is Cloud Computing, Essential Characteristics
- Architectural Influences
- Cloud Delivery Models: SPI Framework (SaaS, PaaS, IaaS)
- Cloud Deployment Models: Public, Community, Private, Hybrid Clouds  
**Hours: 9**

### Module 2: Cloud Architecture and Virtualization
- Exploring Cloud Computing Stack
- Virtualization Technologies, Load Balancing
- Understanding Hypervisors and Machine Imaging
- Cloud Storage Providers, Titans of Cloud (Google, Amazon)
- Accessing the Cloud: Platforms, Web Applications, Web APIs  
**Hours: 9**

### Module 3: Containers and Microservices
- Virtual Machines, Containers, Serverless Computing
- Container Basics, Docker and Docker Hub
- Agents and Microservices, Container Resource Managers
- Identity Management in Swarm
- Amazon EC2 Container Service, Google Kubernetes  
**Hours: 9**

### Module 4: Cloud Management and Security
- SLA Management: Types, Lifecycle, SLA in Cloud
- Cloud Information Security Objectives, Services
- Cloud Security Design Principles
- Secure Cloud Software Requirements
- Privacy & Compliance Risks, Threats, Access Control
- Security Architecture: Identity Management, Access Control  
**Hours: 9**

## reference books

- *Distributed and Cloud Computing: From Parallel Processing to Internet of Things* – Kai Hwang, Geoffrey C. Fox, Jack Dongarra
- *Getting Started with Kubernetes* – Jonathan Baier
