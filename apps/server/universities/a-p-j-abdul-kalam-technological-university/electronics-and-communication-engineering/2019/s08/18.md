# ECT438: Computer Vision

## Course Objectives

- Familiarize students with fundamental image processing operations for computer vision applications.  
- Introduce mathematical morphology and shape analysis techniques.  
- Develop understanding of feature detection, image segmentation, and object localization methods.  
- Explore motion analysis and optical flow estimation techniques.  
- Apply computer vision algorithms to real-world object recognition and scene analysis tasks.  

## Course Outcomes

- Understand digital filtering operations for computer vision applications.  
- Apply basic morphological and boundary operators for computer vision applications.  
- Apply edge and corner detection algorithms to locate objects in an image.  
- Apply optical flow algorithms to detect moving objects in a video.  
- Analyse a given scene using appropriate computer vision algorithms to detect/recognize objects and implement them in real-time practical applications.  

## Course Content

- **Module I — Review of Image Processing Techniques**  
  - Digital filters, linear filters, homomorphic filtering.  
  - Point operators: histogram processing, neighbourhood operators, thresholding.  

- **Module II — Mathematical Morphology and Shape Analysis**  
  - Erosion, dilation, opening, closing, hit-or-miss transform.  
  - Connectedness, object labelling, counting.  
  - Boundary descriptors: chain codes.  
  - Properties of binary regions, geometric features, statistical shape properties.  

- **Module III — Feature Detection and Image Segmentation**  
  - Edge detection: edges, lines, active contours.  
  - Split-and-merge, mean shift, mode finding.  
  - Normalized cuts, graph cuts, energy-based methods, Canny’s algorithm.  
  - Corner detection: Harris corner detection algorithm.  
  - Hough transform for line and curve detection.  

- **Module IV — 3D Scene Understanding and Motion Analysis**  
  - Shape from X: shape from shading, photometric stereo, texture, occluding contour detection.  
  - Motion analysis: regularization theory.  
  - Optical flow: brightness constancy equation, aperture problem, Horn–Schunck method, Lucas–Kanade method.  
  - Structure from motion.  

- **Module V — Object Recognition and Real-time Applications**  
  - Shape correspondence and matching.  
  - PCA, SVM, LDA, Bayes rule, ML methods.  
  - Eigenfaces, face detection, face recognition.  
  - Scene analysis applications.  
  - Example: in-vehicle vision systems.
